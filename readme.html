Prerequisites

This version of the Flexible Footer Menu Multilingual II requires:

  1. A working copy of Zen Cart 2.1.0 or later.
  2. A working installation of the ZCA Bootstrap Template, v3.7.4 or later.
  3. The site must use a PHP version greater than or equal to PHP 8.0.

Overview

Flexible Footer Menu Multilingual II (for Bootstrap) (BFFM) provides an admin-configurable footer menu layout for Zen Cart installations built with the ZCA Bootstrap template.

Note: This plugin is derived from the similarly-named plugin Flexible Footer Menu Multilingual v1.5, created by @rbarbour and @Design75, but its processing is somewhat different.

Installation

The installation steps for BFFM are simple, since the plugin is encapsulated.

  1. Unzip the plugin's zip-file distribution.
  2. Copy the contents for the distribution's zc_plugins directory to your site's /zc_plugins directory.
  3. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  4. You'll see a plugin named Bootstrap: Flexible Footer Menu Multilingual. Click on the associated Install button, confirm the installation and you're finished!
  5. Now that the plugin's installed, refer to this readme's How to Use tab to configure the plugin for your use.
Database Modifications

The plugin's admin installation script makes modifications to your site's database:

  1. Two new tables are added (if not already present):
    1. flexible_footer_content2
    2. flexible_footer_menu2
  2. On an initial install, i.e. the above two tables are added, those tables are initialized depending on:
    1. If a version of Flexible Footer Menu Multiplingual is installed, the tables are initialized from that plugin's flexible_footer_menu_content and flexible_footer_menu tables, respectively.
    2. Otherwise, if a version of the original Flexible Footer Menu is installed, that tables are initialized from that plugin's flexible_footer_menu table.
    3. Otherwise, the tables are initialized with some sample data.
Uninstalling the Plugin

To uninstall the plugin:

  1. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  2. Locate the entry for Bootstrap: Flexible Footer Menu Multilingual, click the associated Un-Install button, confirm the removal and you've removed the plugin.

Note: The database tables added during installation are not removed when the plugin is uninstalled.
